# data-audit
# Audit Automation Tool

## Objective
Automate pre-audit data analysis for internal audit teams.

## Features
- KPI calculation (Revenue, Margin, Volume)
- Risk detection (negative margin, duplicates, anomalies)
- Interactive dashboard (Streamlit)

## Tech Stack
- Python
- Pandas
- Streamlit

## Usage
1. Upload Flash Report (Excel)
2. View dashboard
3. Export risk tables
